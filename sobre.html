Descrição do Projeto:

Este projeto foca na análise e visualização de dados relacionados à pandemia de COVID-19, fornecendo uma visão abrangente sobre a evolução da doença em diferentes regiões e taxas de mortalidade. O objetivo é fornecer informações claras e atualizadas para facilitar a tomada de decisões e a compreensão do impacto global da pandemia.

Objetivo:

O objetivo principal deste projeto é criar uma plataforma interativa que permita aos usuários explorar dados sobre a COVID-19 de maneira acessível e informativa. A plataforma busca ajudar na identificação de padrões, na compreensão da disseminação do vírus.

Tecnologias e Ferramentas Utilizadas:

* HTML , CSS E JAVASCRIPT * Git e GitHub * Covid19 Brazil API * https://paletadecores.com * GOOGLE CHARTS
